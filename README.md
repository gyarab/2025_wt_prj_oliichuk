# 2025_wt_prj_oliichuk

## automatizovany tracker statistik pro ranked hry v valorant

### co dělá
Aplikace se automaticky připojuje k Riot API a po každé ranked hře stahuje herní statistiky. Ukládá specifikovane údaje (kills, deaths, assists, rank changes, atd.) do databáze.

### technologie

- Django framework
- Riot Games API
- SQLite/PostgreSQL databáze
